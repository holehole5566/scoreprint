# ScorePrint — 成績單產生器

把新北校務系統匯出的成績 HTML 轉成可列印的個人成績單小卡。

## 使用方式

1. 用瀏覽器開啟 `index.html`
2. 上傳從校務系統列印儲存的 PDF
3. 填入每位學生的「上次校排」，進退步自動計算
4. 按「預覽成績單」確認排版
5. 按「列印」印出 A4，自行裁切

## 校務系統匯出設定

新北校務系統 → 列印成績，使用以下設定：

| 項目 | 設定 |
|------|------|
| 日常生活 | 不顯示 |
| 學號顯示 | 不顯示 |
| 班級名次 | 全部顯示 |
| 年級名次 | 全部顯示 |
| 總分顯示(定期) | 定期總分 |
| 顯示內容(定期) | 定期 |
| 計算備註(定期) | 顯示加權 |
| 預覽型態 | HTML |

預覽後用瀏覽器「列印 → 儲存為 PDF」存檔，再上傳至 ScorePrint 使用。
